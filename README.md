# Gestionnaire de Contacts

## Description
Programme Java qui gère une liste de contacts avec ajout et recherche.

## Fonctionnement
- Crée un gestionnaire de contacts
- Ajoute 5 contacts avec nom et numéro de téléphone
- Recherche un contact par nom et affiche son numéro

## Comment exécuter le projet
1. Cloner le dépôt :
```bash
https://github.com/inzajrtoure5/Exercice1_ContactsManager
```
2. Ouvrir dans IntelliJ IDEA
3. Lancer `Main.java`

## Prérequis
- Java JDK 21
- IntelliJ IDEA
